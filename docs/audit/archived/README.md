# Archived Audit Reports

Audit reports that are no longer the current authoritative version are
moved here, preserved by the year-month they were superseded.

| Subdirectory | Contents |
| ------------ | -------- |
| `2026-04/` | Reserved for any 2026-04 audit reports superseded after the 2026-05-04 freshness sweep. Currently empty — every section of the live audit reports passed spot-check on 2026-05-04, so nothing needed to be moved. |

When archiving, copy the file (or the relevant section) here with the
original filename and add a one-line note at the top of the live report
pointing here. Do not delete history.
