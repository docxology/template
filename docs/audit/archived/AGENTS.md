# audit/archived/ - Agent Guide

## Purpose

This directory preserves audit reports or report sections that are no longer the
current authoritative snapshot.

## Local Rules

- Do not move living rosters or current findings here.
- Add archival material under a dated subdirectory such as `YYYY-MM/`.
- Keep the live report pointing to the archived copy when material is moved.
- Treat line numbers and project rosters in archived reports as historical
  snapshots, not current facts.
