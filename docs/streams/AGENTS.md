# `docs/streams/` — Stream and Talk Notes

## Purpose

`docs/streams/` stores timestamped notes for livestreams and recorded talks related to this repository.
These files are archival context, not normative build or API documentation.

## Contents

- `README.md` — Navigation index for stream notes.
- `inferant-stream-018-1.md` — Notes for InferAnt stream 018.1.

## Conventions

- File names should be stable and date/episode scoped.
- Notes should link back to canonical docs when referencing current behavior.
- Prefer concise factual summaries over roadmap claims.
