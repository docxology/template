# Open Gauss (fep_lean)

In this repository, **Open Gauss** refers to the **[math-inc/OpenGauss](https://github.com/math-inc/OpenGauss)** Lean assistant CLI (`gauss`), as integrated in **`projects/fep_lean/`** only.

It does **not** refer to Huawei OpenGauss DBMS. The current `fep_lean` Python modules do not use a SQLite session database.

Full detail: [projects/fep_lean/docs/opengauss.md](../projects/fep_lean/docs/opengauss.md).
