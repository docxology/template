# infrastructure/core/config/ - Configuration Helpers

Configuration loading, formatting, schema, and CLI helpers for `infrastructure.core`.
The shared `manuscript/config.yaml` schema includes paper metadata, authors,
publication fields, testing thresholds, render-format toggles, the Stage 02
`analysis.scripts` allowlist, and the `project_config` passthrough mapping for
project-specific settings.

## Files

- `loader.py`
- `formatting.py`
- `queries.py`
- `schema.py`
- `cli.py`
