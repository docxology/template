# infrastructure/validation/cli/ - Validation CLI Documentation

## Purpose

The `infrastructure/validation/cli/` package contains the command-line entry points for markdown, PDF, and repository validation.

## Files

- `main.py` - CLI dispatcher
- `markdown.py` - markdown validation CLI
- `pdf.py` - PDF validation CLI
- `__main__.py` - module entry point
