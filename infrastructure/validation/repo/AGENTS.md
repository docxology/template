# infrastructure/validation/repo/ - Repository Validation Documentation

## Purpose

The `infrastructure/validation/repo/` package contains repository scanning, audit orchestration, issue categorization, and documented-command helpers.

## Files

- `scanner.py` - repository scanner
- `audit_orchestrator.py` - audit orchestration
- `models.py` - repository scan models
- `issue_categorizer.py` - issue categorization
- `known_exceptions.py` - known exception handling
- `_repo_ast.py` - AST helpers
- `_repo_documented_commands.py` - documented command checks
- `_repo_scan_report.py` - repo scan reporting
