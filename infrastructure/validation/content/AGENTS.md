# infrastructure/validation/content/ - Content Validation Documentation

## Purpose

The `infrastructure/validation/content/` package contains validators for PDFs, markdown, and figures.

## Files

- `pdf_validator.py` - PDF text and structure validation
- `markdown_validator.py` - markdown structure validation
- `figure_validator.py` - figure validation
