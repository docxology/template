# `data/` — source-owned evidence declarations

This directory contains the quantitative claim ledger used by the shared
evidence registry. Rows point to the tests, formal checks, or project source
surfaces that own the corresponding manuscript fact; generated reports remain
under `output/`.

Do not add a number merely to silence validation. Add a row only when the
source path exists and the claim is bounded to the local artifact named by the
row. Re-run Stage 04 after changing the ledger.
