# Data

Committed, version-controlled inputs for the exemplar (no runtime downloads).

- `claim_ledger.yaml` — human-declared manuscript claims with their expected
  values/evidence, so headline numbers (e.g. the SIA `max_generations` bound)
  are anchored to a declared claim rather than only to the run's own output.

See [`AGENTS.md`](AGENTS.md) for the ledger schema and the rule that only
project *inputs* — never pipeline outputs — belong in this directory.
