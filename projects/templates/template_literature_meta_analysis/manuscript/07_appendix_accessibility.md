# Appendix C: Accessibility and Provenance

**Figures.** All figures are rendered with a colourblind-safe palette and high-contrast
labels at publication DPI, and each carries a descriptive caption so the visual claims are
recoverable from text alone.

**Provenance.** Every reported number is injected from a committed artifact rather than
typed by hand; an unresolved placeholder is a hard error, so the rendered manuscript can
contain no orphaned or stale figures. The configuration hash and artifact inventory bind
the prose to the exact pipeline run that produced it.

**Honesty.** The default corpus is synthetic and labelled as such; the manuscript does not
present fixture-derived numbers as empirical findings about {{SEARCH_TERM_TITLE}}. Live
findings require a real retrieval run with regenerated artifacts.
