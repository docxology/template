# `src/visualization/` — Agent Guide

Deterministic matplotlib figure generation.

**Contents.** `plots.py` builds every chapter figure; `gallery.py` the format gallery; `_scaffold.py` placeholder generation honoring the `<part_id>_<stem>.png` filename contract.

**Contract.** Figures are generated to `output/figures/`, never committed; reproduced by `scripts/generate_figures.py`.

See the project [`AGENTS.md`](../../AGENTS.md) and [`docs/`](../../docs/) for the full map.
