# Output Conventions

- Page PNGs: `output/figures/storybook_pages/`
- Contact sheet: `output/figures/storybook_contact_sheet.png` (inspection aid,
  not the primary full-page artifact)
- Primary storybook PDF: `output/pdf/the-shape-between.pdf`
- Manifest: `output/data/storybook_manifest.json` (includes page-level alt text
  and contact-sheet provenance)
- Summary: `output/reports/storybook_summary.md`
- Repository copied output: `output/templates/template_storybook/`
