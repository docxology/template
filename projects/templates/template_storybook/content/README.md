# Story Content

`story.yaml` defines the cast and the ordered storybook page list.

Each page has:

- `number`
- `slug`
- `title`
- `scene`
- `overlay_box`
- `palette`
- `text`
- `caption_position` (optional; `"bottom"` by default, or `"top"` to place the caption near
  the top of the page — see the `mega_symbol` page for an example)
