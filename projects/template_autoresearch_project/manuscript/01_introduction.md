# Introduction

AutoResearch systems are useful only when their planning, evidence, and review
surfaces are inspectable. This project keeps those surfaces inside the normal
template lifecycle: tests run first, analysis produces structured artifacts,
rendering hydrates the manuscript, validation checks readiness, and copying
publishes the final outputs.

The exemplar intentionally avoids autonomous execution. It turns a research
brief into a deterministic loop report, then lets the repository's existing
pipeline and validation modules decide whether the outputs are ready for human
review.
