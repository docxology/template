# Blake Bimetallism: Manuscript Sources

This directory contains the root 18-stage Markdown documents comprising the *Blake Bimetallism* synthesis manuscript. It acts as the core scientific payload of the project.

## Document Structure

The manuscript is aggressively modularized for programmatic parsing and rendering.

* `00-01`: Frontmatter and framing of the monetary divide.
* `02a-02d`: The chronological macro-history of 18th Century Finance (1717–1797).
* `03a-03c`: The mythological mapping of Urizenic Materialism and Fourfold Vision.
* `04a-04c`: The historical rupture of 1816 and its topological reduction into Ulro.
* `05a-05e`: The Atlantic Crucible — American bimetallism, *America A Prophecy*, transatlantic comparison, 20th-century fiat consolidation, and the 21st-century sound money rebellion.
* `06`: Conclusion bounding the esoteric with the material.

## Configuration & Bibliography

The generation parameters (such as `1in` document margins, CC BY 4.0 licensing, and LLM review execution) are managed explicitly within `config.yaml`.
All historical quantitative quotes and philosophical framings are traced securely within `references.bib` via Pandoc Citeproc.
